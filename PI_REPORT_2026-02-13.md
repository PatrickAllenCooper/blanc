## Project Overview

BLANC is a benchmark for evaluating foundation models on defeasible abductive reasoning using expert-curated knowledge bases. The project provides:

1. **Expert-grounded knowledge** from 4 peer-reviewed institutions
2. **Complete codec framework** supporting 4 representation modalities
3. **Comprehensive evaluation infrastructure** for testing 5 foundation models
4. **Statistical analysis pipeline** for benchmark validation

- **Evaluation Pipeline**:
  - Response caching (persistent storage, cost savings)
